# Keyboard Shortcuts

This is a comprehensive reference of keyboard shortcuts available in Quicksilver.

## From macOS

| Key combination | What it does |
|---|---|
| <kbd>⌃</kbd><kbd>Space</kbd> | Invoke Quicksilver (default) |
| <kbd>⌘</kbd><kbd>⎋</kbd> | Invoke Quicksilver with Finder or frontmost app selection |

| Key combination | What it does |
|---|---|
| <kbd>↩</kbd> or <kbd>⌃</kbd><kbd>O</kbd> | Execute current command |
| <kbd>⌃</kbd><kbd>↩</kbd> | Collapse current command into first pane as a single item |
| <kbd>⇥</kbd> | Move focus to next pane |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>Letter</kbd> | Execute command for action associated with <kbd>⇧</kbd><kbd>Letter</kbd> |
| <kbd>⌫</kbd> | Clear entered keystrokes in current pane |
| <kbd>⌫</kbd> | Clear selection in current pane if no entered keystrokes |
| <kbd>⌘</kbd><kbd>↩</kbd> | Run the alternate for the current action |
| <kbd>⌃</kbd><kbd>⌘</kbd><kbd>D</kbd> | Set as default item for current search |
| <kbd>⌘</kbd><kbd>K</kbd> | Show task viewer |
| <kbd>⌘</kbd><kbd>L</kbd> | Show Clipboard History (requires Clipboard Plugin) |
| <kbd>⌃</kbd><kbd>⌘</kbd><kbd>Q</kbd> | Restart Quicksilver |
| <kbd>⌘</kbd><kbd>R</kbd> | Rescan catalog |
| <kbd>⌥</kbd><kbd>⌘</kbd><kbd>S</kbd> | Show Shelf (requires Shelf Plugin) |
| <kbd>⌘</kbd><kbd>T</kbd> | Open Fonts |
| <kbd>⇟</kbd> or <kbd>⌃</kbd><kbd>V</kbd> | Next page in list |
| <kbd>⇞</kbd> or <kbd>⌥</kbd><kbd>V</kbd> | Previous page in list |
| <kbd>↓</kbd> or <kbd>⌃</kbd><kbd>N</kbd> or <kbd>⌃</kbd><kbd>J</kbd> | Next item in list |
| <kbd>↑</kbd> or <kbd>⌃</kbd><kbd>P</kbd> or <kbd>⌃</kbd><kbd>K</kbd> | Previous item in list |
| <kbd>⇲</kbd> | End of list |
| <kbd>⇱</kbd> | Start of list |
| <kbd>⌘</kbd><kbd>,</kbd> | Preferences |
| <kbd>⌘</kbd><kbd>?</kbd> | Guide |
| <kbd>⌘</kbd><kbd>\u2019</kbd> or <kbd>⌃</kbd><kbd>⌘</kbd><kbd>:</kbd> | Triggers |
| <kbd>⌘</kbd><kbd>;</kbd> | Catalog |
| <kbd>⌘</kbd><kbd>\u201d</kbd> or <kbd>⇧</kbd><kbd>⌘</kbd><kbd>\u2019</kbd> | Plug-ins |
| <kbd>⎋</kbd> or <kbd>F5</kbd> | Dismiss list if visible |
| <kbd>⎋</kbd> or <kbd>F5</kbd> | Dismiss Quicksilver if no list visible |

## Within the First Pane

| Key combination | What it does |
|---|---|
| <kbd>⎋</kbd> or <kbd>F5</kbd> | Reset search to top level of Catalog if list visible |
| Hold letter | Execute default command for object associated with letter |
| <kbd>~</kbd> or <kbd>`</kbd> or <kbd>⌥</kbd><kbd>⌘</kbd><kbd>N</kbd> | Home folder |
| <kbd>⌘</kbd><kbd>A</kbd> | Select all in list |
| <kbd>⌘</kbd><kbd>G</kbd> | Grab Finder or frontmost app selection |
| <kbd>⌥</kbd><kbd>⌘</kbd><kbd>G</kbd> | Grab \u2019n Drop |
| <kbd>⌘</kbd><kbd>[</kbd> | Go back to the previously selected object (browse history back) |
| <kbd>⌘</kbd><kbd>]</kbd> | Go forward to next selected object (browse history forward) |
| <kbd>→</kbd> or <kbd>/</kbd> or <kbd>⌃</kbd><kbd>L</kbd> | Move down into folder/item |
| <kbd>←</kbd> or <kbd>⇧</kbd><kbd>/</kbd> or <kbd>⌃</kbd><kbd>H</kbd> | Move up out of folder/item |
| <kbd>⌥</kbd><kbd>→</kbd> or <kbd>⌥</kbd><kbd>/</kbd> | Show hidden files |
| <kbd>,</kbd> | Collect items (Comma Trick) |
| <kbd>⌥</kbd><kbd>,</kbd> | Uncollect items (Comma Trick) |
| <kbd>.</kbd> | Enter text mode |
| <kbd>=</kbd> | Calculate mathematical expression (requires Calculator Plugin) |

## Within the Second Pane

| Key combination | What it does |
|---|---|
| Hold letter | Execute command using action associated with letter |

## Spacebar Behavior

The behavior of the spacebar is determined in Preferences \u2192 Preferences \u2192 Command. The table below describes the effect of each option.

| Option | What it does |
|---|---|
| Normal | Returns items with space(s) in the name |
| Select Next Result | Next item in list |
| Jump to Argument Field | Moves focus to pane 3 and subsequently selects the next action that requires pane 3 |
| Switch to Text Mode | Also adds a space to the typed text (pane 1 only) |
| Show Item\u2019s Contents | Drill down into object (pane 1 only) |

## Standard macOS Shortcuts

Standard macOS shortcuts also work in Quicksilver: <kbd>⌘</kbd><kbd>H</kbd> Hide, <kbd>⌘</kbd><kbd>W</kbd> Close, <kbd>⌘</kbd><kbd>Q</kbd> Quit, <kbd>⌘</kbd><kbd>X</kbd> Cut, <kbd>⌘</kbd><kbd>C</kbd> Copy, <kbd>⌘</kbd><kbd>V</kbd> Paste, <kbd>⌘</kbd><kbd>B</kbd> Bold, <kbd>⌘</kbd><kbd>U</kbd> Underline, <kbd>⌥</kbd><kbd>⌘</kbd><kbd>T</kbd> Show Character Palette.

Some shortcuts, such as the main invocation hotkey, can be modified in Preferences, and others can be added with Triggers.

Note that system-wide shortcuts (such as <kbd>⌘</kbd><kbd>⎋</kbd> to enter Front Row) might supersede application shortcuts\u2014you may have to disable these in System Preferences to expose the Quicksilver key combination.
